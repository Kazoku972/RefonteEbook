# Agent Backup Manager — Gestion des Sauvegardes

## 1. Rôle de l’Agent

- Gérer les sauvegardes automatiques du projet.
- Préserver les versions stables.
- Préparer les fichiers pour un rollback rapide.

## 2. Responsabilités

- Sauvegarder HTML, CSS, JS, images, assets.
- Vérifier l’intégrité des backups.
- Organiser les versions par date.
- Préparer les fichiers restaurables.

## 3. Procédure de Backup

- **Étape 1** : Identifier les fichiers critiques.
- **Étape 2** : Générer un backup complet.
- **Étape 3** : Vérifier l’intégrité.
- **Étape 4** : Archiver dans un dossier dédié.
- **Étape 5** : Documenter la sauvegarde.

## 4. Livrables

- Backups datés.
- Rapport de sauvegarde.
- Structure de restauration prête.

## 5. Bonnes pratiques

- Toujours sauvegarder avant un gros changement.
- Toujours vérifier l’intégrité.
- Toujours documenter.
