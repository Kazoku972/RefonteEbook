# Ebook Flipbook - Déploiement GitHub Pages

Ce dossier contient la version finale et optimisée du flipbook interactive "On est ensemble, Maman".

## Structure

- `ebook_flipbook.backup.html` : Fichier principal (Point d'entrée).
- `css/` : Contient les styles interactifs.
- `js/` : Contient les moteurs jQuery et Turn.js (version 3).
- `images/` : Contient tous les visuels et textures papier.

## Caractéristiques

- **Mode Offline** : Toutes les ressources sont locales.
- **Responsive** : Mode single page forcé sous 900px pour mobile et tablette.
- **Tactile** : Support swipe natif.
